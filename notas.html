Permiten referenciar elementos del HTML para manipularlos desde un archivo CSS o JS

    class

    Permite seleccionar elementos por clase. Esto permite referenciar a grupos especificos de elementos, no necesariamente del mismo tipo.

    A cualquier etiqueta que luego le agreguemos un ".nombreDeClase" le asignaremos una clase.
    Ej: "h1.juan" --> 



    id

    Es similar al class pero funciona para un unico elemento


    etiqueta

    Permite referenciar a todas las etiquetas del mismo tipo. Por ejemplo, si en CSS pongo h1{} modificaria todos los h1.
